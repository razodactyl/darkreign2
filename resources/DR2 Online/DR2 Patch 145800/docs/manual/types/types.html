Type Configuration Reference

CreateObjectType(string name, string codeclass)
name: Name of the type. Used when being refered to by other type definitions.
codeclass: Code specific class. The code class defines which of the following sub scopes are requied. The quoted names are the available code classes.
  • GameObj()
    • MapObj()
      • UnitObj() "Unit"
          RestoreObj() "Restore"
      • PropObj() "Prop"
      • ResourceObj() "Resource"
      • ProjectileObj() "Projectile"
      • ExplosionObj() "Explosion"
    • ObjectiveObj() "Objective"
