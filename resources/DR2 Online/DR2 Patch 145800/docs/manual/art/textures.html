Creating Textures

Textures

Texture heights and widths must be integral powers of 2:  32, 64, 128, 256.

No bigger than 256 and square for last generation cards.
But we can auto downsample to correct?

Opaque and translucent textures shouldn't be mixed on one sheet; otherwise, sheeting is a good idea.

Alpha channels aren't necessary for translucent textures that are rendered with the 'glow' blending mode.

Translucent textures that fade to 0 alpha need a 3 pixel border of 0 alpha to allow for mipping, filtering, etc.
