Interface Commands

Activate(string control)
Multiple, control Optional (Current control)
control: name of a control.
Activates a control.
Deactivate(string control)
Multiple, control Optional (Current control)
control: name of a control.
Deactivates a control.
ToggleActive(string control)
Multiple, control Optional (Current control)
control: name of a control.
Activates a deactivated control, deactivates an activated control.
Notify(string control, string event)
Multiple, control Required, event Required
control: name of a control.
event: name of an event.
Sends an event to the control.
NotifyParent(string event)
Multiple, event Required
event: name of an event.
Sends an event to the parent control.
Cmd(string cmd)
Multiple, cmd Required
cmd: A console command
Executes a console command.
IFaceCmd(string cmd, ...)
Multiple, cmd Required
cmd: A var, local var, or console command
Constructs a Cmd by substituting dynamic var names for their fully qualified paths.
