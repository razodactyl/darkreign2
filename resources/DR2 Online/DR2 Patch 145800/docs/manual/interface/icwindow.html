Window

Simple windowing functionality

Functions
None
Style Attributes
TitleBar - Adds a title bar
ThinTitleBar - Umm, same as the title bar, but thinner!
CloseButton - Add a close button (on by default)
NoSysButtons - Do not place any system buttons on the window
AdjustWindow - Resize the window to make the client the specified size
