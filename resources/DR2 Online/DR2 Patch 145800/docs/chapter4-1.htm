4.1 Making Multiplayer Maps

This section will take you through the steps of creating a playable, standard multiplayer map for Dark Reign 2.
(note: this order is not necessarily set in stone)

1. Create a New map, choose a Size and World (Section 3.1 Tools)
2. Create the map's terrain (Section 2.3 Terrain Menu)
3. Add water if map requires water (Section 2.4 Water Menu)
4. Texture and color the map (Section 2.1 Terrain Menu, 2.3 Terrain Color Menu, 2.10 Overlay menu)
5. Place ambient objects (Section 2.2 Objects Menu).
6. Configure start locations, ruleset, other misc settings (described in detail below)

Start locations, ruleset, other misc settings:
Make 1 team for each player the map will support. To create a team, see Section 3.2. For example, for a 4 player map, you would create 4 teams and give them each a unique name (the names don't matter):

TEAM1
TEAM2
TEAM3
TEAM4

Now we are going to 'mark' the spots that are going to be the player's start locations. Hit 5 to get the Region menu. Left-click and drag on the terrain to create a region. It will look like a box. Make a small region at each start location (define the box, type a name in the field just above the "create" button, then click "create". Your new Region should now appear in the list on the right of the screen.) The center of the Region will be where the camera is looking at the start of the game, it will also be the spot where your starting rigs are spawned in. To keep things simple, name the regions the same as your team names. So I made one region at each start loc called TEAM1, TEAM2, TEAM3, and TEAM4.

Now, you'll need to go into each team's config and make some settings. Double click on the team name to get the team config menu. For EACH team do the following:

Make sure the following buttons are depressed (highlighted):
Default Team (just the first team created)
Avail To Players
Show Stats

Make sure the following buttons are NOT depressed:
Permanent Radar
Requires AI
Fixed Side
Enable Type filtering

Set starting resource amount in the "Resource Store" window.

Set start angle in the Angle window (-45 is North)

Set starting locations with the "Start Camera" window. (click the "browse" button next to the Start Camera window. Specify the start regions you made for each team. i.e. in TEAM1's config, select the region TEAM1, etc)

Finally, you need to set a default ruleset for the map. All of the standard multiplayer maps in DR2 have "Protect HQ" as the default ruleset. Click "Tools" in the lower left. Select "Mission". Select "Protect HQ" from the pull down box. (Make sure "Fixed Rule" is NOT depressed)

Save map with CTRL+S. Save in the "Standard Multiplayer Missions" folder, and your map should now show up on the menu when you run the game! Before distributing your map you should use the PAK utilities to zip it up! (see editor_readme.txt on your CD).
